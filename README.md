# Volt Runner

Volt Runner is a small browser mini-game built with plain HTML, CSS, and JavaScript. It is designed to run as a static site, so it can be published directly with GitHub Pages.

## Play locally

Open `index.html` in a browser.

## Project files

- `index.html` contains the game shell.
- `styles.css` contains the responsive layout and visual design.
- `game.js` contains the board logic, levels, rendering, and input handling.

## GitHub Pages

After pushing this folder to GitHub, enable Pages from the repository settings and choose the `main` branch with `/root` as the source.
